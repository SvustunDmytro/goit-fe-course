Строчные и Блочные элементы html.

Строчные элементы.

  • Тег <a> - Является одним из важных элементов HTML и предназначен для создания ссылок.
  • Тег <b> - Устанавливает жирное начертание шрифта.
  • Тег <br> - Перенос строки.
  • Тег <i> - Устанавливает курсивное начертание шрифта.
  • Тег <input> - Позволяет создавать разные элементы интерфейса.
  • Тег <label> - Устанавливает связь между определенной меткой и элементом формы.
  • Тег <textarea> - Создает поле для многострочного текста.
  • Тег <span> - Универсальный строчный элемент.
  • Тег <button> - Создает на веб-странице кнопки.

Блочные элементы

  • Тег <div> - Универсальный блочный элемент.
  • Тег <form> - Устанавливает форму на веб-странице.
  • Тег <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - Заголовоки 1,2,3,4,5,6-го уровня.
  • Тег <ol> - Устанавливает нумерованный список.
  • Тег <ul> - Устанавливает маркированный список.
  • Тег <p> - Определяет текстовый абзац.
  • Тег <table> - Создает таблицу.
  • Тег <article> - Задает содержание сайта вроде новости, статьи, записи блога, форума или др.
  • Тег <section> - Задаёт раздел документа, может применяться для блока новостей, контактной информации, глав текста, вкладок в диалоговом окне и др.
  • Тег <header> - Задает «шапку» сайта или раздела, в которой обычно располагается заголовок.
  • Тег <footer> - Задаёт «подвал» сайта или раздела, в нём может располагаться имя автора, дата документа, контактная и правовая информация.
